What Happened?

If you look at the source code for this page, you'll see that the basic html for the sections above is exactly the same as for the previous page: same h1, h2, and p tags, with no new information in the markup. The only change is the addition of this line to the "head" section of the page:

<link rel="stylesheet" href="style1.css" type="text/css" />

This line causes the page to link to a stylesheet, contained within the same folder as the page, called "style1.css". That stylesheet contains information about how the results of various tags should look in your browser.
