Basic CSS

Like HTML files, CSS files are plain text, editable in any text editor. (As I attempted to show you last week, I like TextWrangler, because of the way it formats your code, making your tags visible.) A CSS file is always saved with the file extension ".css".

Here's the basic format of CSS code:

p {
    color: red;
}

This instructs your browser to make all text contained within <p> tags in any page linked to this stylesheet red.

CSS Structure

Note the structure of the CSS entry above. It begins with "p", which is the tag whose output you want to define, followed by an open curly bracket. The entry also ends with a close curly bracket. The brackets are absolutely required, as they tell the system that everything inbetween belongs to "p".

Inbetween the curly brackets, you list the properties you want to define, and the values for each of those properties, in this form:

property: value;

Properties include, as in our example above, things like color; there are also properties for setting the font that your text uses, the size of the font, the spacing around the text, and many other things besides.

Note that the property name must be followed by a colon, and that each value must be followed by a semi-colon. You can use multiple properties to control multiple aspects of your tags, like this:

p {
    color: red;
    font-size: 14px;
}
