Creating Pop-Up Windows in Dreamweaver 2.0

Step 1

Download and install a free demo version of Dreamweaver 2.0 from Macromedia's web site.
Step 2

Choose File > New Site. You'll see a box that offers you several options. For now, let's name the site Pop-Up Practice.

Important: A site is not the same thing as a document or file. Go to the Help menu and select Dreamweaver Help Files for clarification and for help getting started.

Step 3

Let's create a base document with two annotations: one to a document you wrote, and one to a document posted elsewhere on the Web. In order to do that, you need to create and save a document that will serve as your base page. Choose File > New. Type the following text into your document:

The quick black dogs jumped over the lazy cat.

Save the page as base.html and leave it open.
Step 4

Now we need to create our annotation. Again, choose File > New. Type the following text:

In this sentence, the "quick black dogs" are my dogs, Ivy and Napa.

Save the page as note.html and close the document.
Step 5

Go back to base.html and highlight the words "quick black dogs." Now here's where it starts to get a little tricky: we're going to make "quick black dogs" a link, but it will link to the base document rather than to a separate document! Do this by selecting Window > Properties and entering base.html in the Link box. In the Target box, select _self.

Step 6

Highlight "quick black dogs" again. Go to Window > Behaviors to bring up the Behaviors box..

Click the plus sign to see which behaviors you can associate with the text you have highlighted. Choose Open Browser Window.
Step 7

Now you need to specify which document the pop-up window will bring up and what it should look like. Since you want the window to contain your annotation, enter note.html in the URL box. In the Window Width and Window Height boxes, enter 300. You can change these figures around to be as large or as small as you want.

Finally, check the Scrollbars as Needed box. This lets users scroll down to see text that doesn't automatically fit into the pop-up window. If you don't check it, they won't be able to see it! I also like to check the Resize Handles option, which lets users change the width and height of the window at the same time.

When you're done, click OK. Remember, you can change these settings later by going to the Behaviors box and clicking on the behavior you have defined.
Step 8 And what if you want someone else's WWW document to appear in your pop-up window? Go back to Step 5 and select some text for your link--let's highlight the word "cat." As in Step 5, enter base.html in the Link box and _self in the Target box.
Step 9 Repeat Step 6, substituting "cat" for "quick black dogs."
Step 10

In the Open Browser Window box, enter the URL for the document you want to appear in the window. For our current purposes, try:

http://www.cyberg8t.com/shasta/leo.jpg

Check the Scrollbars as Needed box and adjust other settings as needed.
Step 11 Save your web page and test it out by going to File > Preview in Browser!
