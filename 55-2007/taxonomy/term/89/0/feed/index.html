Traditionally, an author will introduce a character using both first and last names, and then use just one throughout the novel, usually the last name. Pynchon instead uses many names for each character. Slopthrop, Tyrone, Rocketman, Raketenmensch, etc. Greta, Margherita, Mrs. Erddman. Enzian, Oberst, (that name that starts with an n that I cannot remember). At the end of the novel there is a lot of commentary about the fragmentation of Slothrop.
