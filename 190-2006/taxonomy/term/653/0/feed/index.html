It sets the groundwork for a richly creative society, but remains subservient to the value of creativity. The current debate has this turned around. We have become so concerned with protecting the instrument that we are losing sight of the value. </p> <p>I think that what this is saying is that we usually view intellectual property as a a free source from which other creative thought and direction can be born.
