How to make your Mac much, much better

Submitted by oneoutofseven on 14 November 2006 - 2:15pm.
  • apple
  • mac
  • software

I thought since all of the latest blog entries I've made have only been working towards revealing my inner nerdiness, I would continue this trend by making a list of my OS X programs I've found outside of Apple's offerings.

Adium is a chat client that aims to compete with Apple's iChat. What has always bothered me about iChat is the inability to control the order of displayed contacts; I believe the only two options iChat offers are alphabetical and an organization based on who is available, idle, away, offline, etc. Adium allows you to organize contacts however you want, and allows you to access your AIM, ICQ, .MacIM, Jabber, Yahoo!, and GTalk buddy lists in the same place. You can control how chats are displayed (color schemes, styles [thankfully no chat bubbles!] etc), and you can have all of your chats displayed in one tabbed Safari-like browser that you can just tab through. The program also keeps automatically saves chat transcripts, something that has been really useful for me when I've lost phone numbers or meeting times/dates...

Yojimbo is a program that allows you to keep and organize your notes, web receipts, PDF files, passwords, serial numbers and offline web sites all in one place. All items are kept in a general library, and you can further organize them by adding tags and creating categories. You can also encrypt any information you'd rather not be easily accessible. While you're using any other application you can hit the Quick Input shortcut and you can input any sort of information in the window that pops up. Basically this progrma is ridiculously intuitive, useful and convenient; you may be able to live without it now, but once you start Yojimbo, you won't know how you functioned without it.

PulpFiction is a program that aggregates and organizes RSS- and Atom-feed blog posts from any number of blogs. With the program, you can tag, save and flag important blog posts inside of one interface. It's really easy to set up, automatically updates... I used it a lot last semester, but eventually became so overwhelmed by the backlog I eventually acquired when I got busy that I had to quit. PulpFiction Lite is the free version-- I'm not sure how much the paid version differs, but I couldn't ask for more than I got in the Lite version.

Acquisition is I think the best p2p sharing software available for Macs. The program is organized like iTunes is-- when results surface, you browse them in a very similar way to iTunes browsing feature. Of course, iTunes and Acquisition are fully integrated, and songs you download are automatically added to your iTunes library. The program also supports torrent files, making rough-around-the-edges bit torrent programs like Azureus seem a whole lot less appealing. All of your past searches are saved to a sidebar so that all you have to do is hit the small magnifying glass/Spotlight icon next to them to search for them again. I've generally had more success with Acquisition than other file sharing software programs for Macs.

There are a lot of smaller programs/plug-ins I use to make my Mac a little more user-friendly... Growl provides small notification windows when new emails are received (and gives the sender and subject so you don't have to switch over to Mail every time), downloads are completed, updates are available, etc. AcidSearch is a Safari plug-in that makes the Safari search feature a whole lot more powerful, enabling you to site-search, image-search, and search sites like dictionary,com, international Google, newspapers and shopping sites.

If you guys have any other favorite apps you should post them because I looove testing out new ones!
